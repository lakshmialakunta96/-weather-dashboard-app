# Weather Dashboard

## Project Description
A Weather Dashboard application developed as part of the Thiranex Web Development Internship.

## Features
- Search Weather by City Name
- Fetch API Usage
- Async/Await
- JSON Data Parsing
- Error Handling
- Live Temperature, Humidity and Wind Speed

## Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeather API

## Files Included
- index.html
- style.css
- script.js
